title: "Find and Install Extensions in Your App"
description: "Browse, search, and install Keiyoushi extensions inside Mihon or a Tachiyomi fork. Learn how to filter by language, update, and uninstall sources."
---

After you have [added the Keiyoushi repository](/getting-started/adding-the-repo) to your app, you can install extensions directly from within the app. Each extension you install adds one or more reading sources to your **Browse** tab.

## Install an extension

<Steps>
  <Step title="Open the Extensions screen">
    Tap **Browse** in the bottom navigation bar, then tap **Extensions** at the top of the screen.
  </Step>
  <Step title="Filter or search for the extension you want">
    Use the language filter to narrow the list to extensions in your preferred language. You can also tap the search icon and type the name of a source or site to find it directly.
  </Step>
  <Step title="Tap Install">
    Tap the **Install** button next to the extension you want. Your app will download and install the APK. If Android prompts you to allow installation from unknown sources, grant the permission to continue.
  </Step>
  <Step title="Open the new source">
    Once installed, the extension's sources appear in **Browse → Sources**. Tap a source to start searching or browsing its catalogue.
  </Step>
</Steps>

## Update extensions

When a newer version of an extension is available, the Extensions screen shows an **Update** button next to it. Tap **Update** to install the new version. You can also tap **Update all** at the top of the screen to update every extension at once.

<Tip>
  Keep your extensions up to date. Source websites change their structure over time, and outdated extensions may stop loading chapters or search results correctly.
</Tip>

## Uninstall an extension

To remove an extension, tap it on the Extensions screen to open its detail view, then tap **Uninstall**. Removing an extension also removes all sources it provided from your **Browse** tab, but it does not delete any manga you have already added to your library from those sources.

## NSFW extensions

Extensions that connect to sites with adult content are hidden by default. To make them visible on the Extensions screen, go to **Settings → Browse** and enable **Show NSFW sources**.

<Note>
  Enabling NSFW sources only makes those extensions visible so you can install them. It does not change any content filtering within the sources themselves.
</Note>

## Extensions without an app

If you are not using Mihon or a supported Tachiyomi fork, you can download APK files directly from the [Keiyoushi extensions listing page](https://keiyoushi.github.io/extensions/) and sideload them manually onto your device.
